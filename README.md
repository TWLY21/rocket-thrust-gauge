# Rocket Thrust Gauge

A single-file HTML canvas demo showing a sleek animated rocket thrust gauge.

## Features

- 800x800 canvas display
- smooth thrust bar animation from 0% to 100%
- blue to orange to red color transition
- max-thrust glow, shake, and particle effects
- automatic cooldown and reset loop

## Run

Open `index.html` in a browser.
